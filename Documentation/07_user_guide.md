## Student Guide

### Getting Started

1. **Log In**
   - Navigate to the Secure Fair website
   - Enter your institutional email and password
   - Click "Iniciar Sesión"

### Step 1: Register for a Time Slot

Before you can enroll in a project, you must register for a time slot to attend the fair.

1. Go to "Franjas Horarias" in the student dashboard
2. View available time slots with capacity information
3. Select a slot that fits your schedule
4. Click "Registrar" to confirm
5. You will receive a confirmation

**Important**: You can only register for one time slot per fair period.

### Step 2: Get Your QR Code

After registering for a slot:

1. Go to "Mi Código QR" section
2. Your QR code will be displayed
3. Options:
   - Save the QR code image to your phone
   - Take a screenshot
   - Print it out

**Note**: Keep your QR code ready for when you attend the fair.

### Step 3: Check-In at the Fair

When you arrive at the fair:

1. Go to the entrance/check-in desk
2. Show your QR code to the staff
3. Staff will scan or manually enter your code
4. Wait for confirmation of successful check-in

**Important**: You MUST check in to be able to enroll in projects. Without check-in, you cannot redeem enrollment codes.

### Step 4: Explore Projects

Once checked in:

1. Walk around the fair
2. Visit different project stands
3. Talk to socioformadores about their projects
4. Ask questions and learn about opportunities

### Step 5: Enroll in a Project

When you've decided on a project:

1. Ask the socioformador to generate an enrollment code
2. The socioformador will show you a 6-8 character code on their screen
3. In your student dashboard, go to "Inscripción"
4. Enter the code exactly as shown (case-sensitive)
5. Read and accept the project rules
6. Click "Inscribirse"
7. You will receive a signed enrollment receipt

**Important Notes**:
- Codes expire after 2 minutes
- Each code can only be used once
- You can only enroll in ONE project per fair period
- You must be checked in to enroll

### View Your Enrollment

After enrolling:

1. Go to "Mi Inscripción" section
2. View your enrolled project details
3. See your digitally signed enrollment receipt
4. Download receipt for your records

### What If...

**Q: I registered for a slot but can't attend?**
A: Contact the Social Service office to cancel your registration.

**Q: The code expired before I could enter it?**
A: Ask the socioformador to generate a new code.

**Q: I tried to enroll but got an error?**
A: Common reasons:
- You haven't checked in yet
- The code is invalid or expired
- You're already enrolled in another project
- The project is at full capacity

**Q: I lost my enrollment receipt?**
A: You can always view it in your dashboard under "Mi Inscripción"

## Socioformador Guide

### Getting Started

1. **Log In**
   - Use your assigned socioformador credentials
   - Navigate to Secure Fair website
   - Enter email and password

### View Your Projects

After logging in:

1. You'll see a list of projects assigned to you
2. Each project shows:
   - Project name and organization
   - Current enrollment count
   - Maximum capacity
   - Fill rate

### Generate Enrollment Code

When a student wants to enroll:

1. Make sure the student is present at your stand
2. Click "Generar Código" button on your project card
3. A random code will appear on screen (e.g., "XK7M9P")
4. Show this code to the student
5. The student enters it in their app
6. The code expires after 2 minutes

**Best Practices**:
- Only generate codes for students physically present
- Show the code clearly to the student
- Don't share codes via text/email/photos
- Generate a new code if the previous one expired

### Monitor Enrollments

Real-time enrollment tracking:

1. View current enrollment count on your project card
2. The counter updates instantly when students enroll
3. You'll see a warning when approaching capacity

### View Enrolled Students

To see who has enrolled:

1. Click on your project name
2. View list of enrolled students with:
   - Name
   - Student ID (matricula)
   - Email
   - Phone number
   - Enrollment date/time

### Export Student List

To download your student list:

1. Click "Exportar" button on your project
2. Choose format (CSV or Excel)
3. File downloads automatically
4. Use this list for your records and communication

### Tips for Success

- **Keep your device charged**: You'll need it throughout the fair
- **Be welcoming**: Explain your project clearly to interested students
- **Monitor capacity**: Watch your enrollment count
- **Verify presence**: Only generate codes for students at your stand
- **Save exports**: Download student lists for follow-up

## Administrator Guide

### Getting Started

Log in with administrator credentials to access the full admin panel.

### Manage Fair Periods

#### Create a Fair Period

1. Navigate to "Períodos" section
2. Click "Nuevo Período"
3. Fill in:
   - Period name (e.g., "Feb-Jun 2026")
   - Start date/time
   - End date/time
   - Active status
4. Click "Guardar"

#### Edit or Delete Period

- Click edit icon to modify
- Click delete icon to remove (careful: cascading delete)

### Manage Organizations

#### Add Organization

1. Go to "Organizaciones"
2. Click "Nueva Organización"
3. Enter organization name
4. Save

Organizations are the socioformador entities (NGOs, government agencies, etc.)

### Manage Projects

#### Create Project

1. Go to "Proyectos"
2. Click "Nuevo Proyecto"
3. Fill in details:
   - Project name
   - Organization (select from dropdown)
   - Fair period
   - Description
   - Rules text (students must accept these)
   - Capacity (leave empty for unlimited)
   - Assigned socioformadores (select users)
4. Save

#### Edit Project

- Modify any field including capacity
- Add/remove assigned socioformadores
- Activate/deactivate projects

### Manage Time Slots

#### Create Time Slot

1. Go to "Franjas Horarias"
2. Click "Nueva Franja"
3. Configure:
   - Fair period
   - Start date/time
   - End date/time
   - Capacity (max students)
   - Location
4. Save

**Tips**:
- Create slots to distribute student arrivals
- Typical slot: 2 hours
- Consider venue capacity when setting limits

### Perform Check-In

On fair day at the entrance:

1. Navigate to "Check-in" section
2. Options:
   - **Scan QR code**: Use device camera to scan student QR
   - **Manual entry**: Student provides code, you type it in
3. Click "Verificar"
4. System confirms:
   - Student name
   - Student ID
   - Slot information
5. Student is now verified and can enroll

**Troubleshooting**:
- Invalid QR: Ask student to regenerate from their dashboard
- Already checked in: Student can proceed to fair
- No slot registration: Student must register for slot first

### Monitor Dashboard

Real-time analytics:

1. Go to "Dashboard"
2. Select fair period
3. View metrics:
   - Total slots created
   - Total registrations
   - Total check-ins
   - Total enrollments
4. Charts:
   - Attendance by slot
   - Enrollment by project
   - Capacity fill rates

### Export Data

#### Master Export

Complete data export:

1. Go to "Exportaciones"
2. Select "Exportación Maestra"
3. Choose period
4. Select format (CSV/Excel)
5. Click "Exportar"

Export includes:
- Student information
- Slot registration
- Check-in status
- Project enrollment
- Organization details

#### Per-Project Export

Export for specific project:

1. Navigate to project details
2. Click "Exportar"
3. Get list of enrolled students

### User Management

#### Create Users

1. Go to "Usuarios"
2. Click "Nuevo Usuario"
3. Fill in:
   - Email
   - Password
   - Role (ADMIN, SOCIO, STUDENT)
4. For students, additional fields:
   - Matricula
   - Full name
   - Phone

#### Assign Socioformadores to Projects

1. Edit project
2. In "Socioformadores Asignados" section
3. Select users with SOCIO role
4. Save

### Import Data (Bulk Operations)

For large-scale setup:

1. Download Excel template
2. Fill in organizations, projects, or slots
3. Upload file
4. Review preview
5. Confirm import

**Note**: Validate data carefully before import.

### Best Practices for Admins

- **Pre-Fair Setup**:
  - Create period 2-3 weeks in advance
  - Set up organizations and projects early
  - Create time slots to distribute load
  - Test check-in process before fair day

- **During Fair**:
  - Have multiple check-in stations
  - Monitor dashboard for capacity issues
  - Be ready to assist socioformadores
  - Keep backup of QR scanner devices

- **Post-Fair**:
  - Export master data immediately
  - Archive fair period
  - Gather feedback for improvements

## FAQ

### General Questions

**Q: What browsers are supported?**
A: Chrome, Firefox, Safari, Edge (latest versions)

**Q: Is there a mobile app?**
A: No native app, but the website is mobile-responsive

**Q: What if I forget my password?**
A: Contact the Social Service administrator for password reset

### Student FAQs

**Q: Can I change my time slot after registering?**
A: Contact administration; generally slots are fixed

**Q: What if I arrive late to my slot?**
A: You may still check in if the slot is still active

**Q: Can I enroll in multiple projects?**
A: No, only ONE project per fair period

**Q: What if my preferred project is full?**
A: Choose another project; capacities are enforced

**Q: How do I know my enrollment is official?**
A: You receive a digitally signed receipt; also visible in your dashboard

### Socioformador FAQs

**Q: A student has my code but isn't at my stand. Can they enroll?**
A: No, the system requires physical check-in first

**Q: What if I accidentally generate a code?**
A: The code will expire in 2 minutes if unused

**Q: Can I see students who are interested but not enrolled?**
A: No, only enrolled students appear in your list

**Q: What happens if we reach capacity?**
A: System automatically prevents more enrollments

### Admin FAQs

**Q: Can I delete a fair period with data?**
A: Yes, but use caution (cascading delete). Export data first!

**Q: How do I handle duplicate student registrations?**
A: Database constraints prevent duplicates automatically

**Q: What if check-in device fails?**
A: Have backup device ready; or manual entry option available

**Q: Can I extend a project's capacity mid-fair?**
A: Yes, edit the project and increase capacity
